# G-TALKS — Site Starter

Bu repoda G-TALKS konferans sitesi için basit bir statik starter bulunmaktadır. Benzer içerikleri doldurup GitHub Pages veya Netlify/Vercel ile hızlıca yayınlayabilirsiniz.

Dosya yapısı:
- index.html
- styles.css
- script.js
- sitemap.xml
- robots.txt
- assets/
  - g-talks-logo.svg (placeholder logo — isteğe göre png/svg ile değiştirin)
  - g-talks-favicon.svg (favicon)
  - speaker1.svg, speaker2.svg (konuşmacı görsel placeholder)

Hızlı deploy (GitHub Pages):
1. Repo'yu klonlayın veya dosyaları push edin.
2. Settings → Pages bölümünden Branch: `main`, Folder: `/ (root)` seçin.
3. Birkaç dakika sonra site: `https://{kullanici}.github.io/G-Talks-/` adresinde yayında olur.

Notlar:
- Kendi logo PNG/SVG dosyanızı `assets/g-talks-logo.svg` yerine koyun veya index.html içindeki yolu güncelleyin.
- Konuşmacı fotoğrafları için `assets/` içine gerçek görsellerinizi ekleyin ve script.js içindeki `speakers` dizisini güncelleyin.
- Yayına aldıktan sonra Google Search Console'a siteyi ekleyip sitemap.xml gönderin.
